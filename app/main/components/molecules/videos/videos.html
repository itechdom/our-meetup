Videos

The video is embedded with Brightcove Videocloud. The embed code is taken from their examples, with 2 modifications/conditions:

  • Their JavaScript library is included in the HTML page outside this sample. The protocol-relative URL (which should be used in all cases) to that library is //admin.brightcove.com/js/BrightcoveExperiences.js.
  • Once the embed HTML is on the page, the following JavaScript function should be called to initialize the video: window.brightcove.createExperiences().

The following Flash parameters in the code sample must be replaced:

  • playerID
  • playerKey
  • @videoPlayer
