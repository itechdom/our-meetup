Tables

The tables styling is quite basic with the exception of the color accent. This will be the only change from table to table. You can use the following class names for color accents:

  • bg-gold
  • bg-blue
  • bg-grey
  • bg-black
  • bg-light-grey

The table behaves the same for all breakpoints except Mobile which does not have a standardized version yet.

The tables are still a work in progress.
