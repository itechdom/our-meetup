About the guide

Technology

This guide is a single-page application powered by AngularJS. AngularJS is not required to utilize the design implementations provided in this guide, it simply powers the guide wrapper.

Libraries

Libraries used in the guide:

  • AngularJS 1.2 (Angular 1.3 doesn't support IE8)
  • Angular UI Router - keeps track of different pages in guide and manages navigation between them without page reload
  • Angular Bootstrap - Bootstrap components written for AngularJS (used for Tabs)
  • ng-load - Lets us resize iframes when internal content fully downloads

Libraries used in the guide and in the implementation:

  • jQuery 1.11 - using version 1.11 because 2.0 doesn't support IE8
  • ES5 Shim - used so IE8 knows how to treat newer web features
  • JSON3 - used so IE8 knows how to handle JSON
  • Normalize.css - normalizes all built-in browser styles to avoid cross-browser differences
  • bootstrap-sass-official - The Bootstrap CSS framework, compatable with SASS, our CSS preprocessor of choice
  • Respond.js - Enables media queries in IE8 so appropriate styles can be displayed at different widths

Libraries used only in the implementation:

  • jQuery.mmenu - Used for slide-out menu in the top navigation
  • slick-carousel - Used for carousel
  • bootstrap-datepicker - Used for datepicker
  • modular-scale - SASS library to help size text appropriately
