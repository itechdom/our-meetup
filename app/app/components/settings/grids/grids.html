The Grid

The grid is built through a SASS function that builds the columns, gutters, and offsets. It uses variables inside of the variables.scss sheet to set the number of columns, max-width of the grid, and gutters.

The grid is built using Bootstraps grid system with a few breakpoint modifications:

  • Mobile – 320px to 639px – Bootstrap size xs
  • Tablet – 640px to 969px – Bootstrap size sm
  • Desktop – 970px and up – Bootstrap size md and lg
