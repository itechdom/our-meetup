Forms

Filling out forms is especially painful on mobile devices. Some things to keep in mind to keep your interface user-friendly:

Minimize the amount of data the user needs to input. Remove or combine fields.

When possible. set sensible defaults. For example, default the date on a movie ticket buying site to today.

If keyboard input is required, automatically present the right keyboard. For example, present the number keyboard when the user taps into the ZIP field.
