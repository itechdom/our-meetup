How to use the guide

Example elements

Elements are the building blocks of pages. Each page is comprised of multiple elements brought together. The Layouts section shows examples of component combinations.

Each element can be viewed at desktop, tablet, or mobile widths. This lets you see how the elements change at different page widths. You can open each example in a new window to view it by itself (as opposed to in the frame).

The code tab shows the HTML that powers each element, which can be copied into new pages.

Steps for creating a site

The initial implementation into your site or application will require the following steps. This will include creating the base layout and including required CSS and Scripts to have a base to build on.

  1. Use this HTML template: 
    <!doctype html>
<!--[if IE 8]><html class="ie8"><![endif]-->
<!--[if IE 9]><html class="ie9"><![endif]-->
<!--[if gt IE 9]><html><![endif]-->
<!--[if !IE]> --><html><!-- <![endif]-->
  <head>
    <meta charset="utf-8">
    <title>[[Page Title Here]]</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0, user-scalable=0, maximum-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="apple-mobile-web-app-capable" content="yes">


    <link rel="stylesheet" type="text/css" href="//cloud.typography.com/6666452/703066/css/fonts.css" />
    <link rel="stylesheet" href="styles/vendor.css">
    <link rel="stylesheet" href="styles/global.css">
    <!-- Additional styles below here -->
  </head>
  <body>
    <!--[if lt IE 8]>
    <p>[[Message for users in outdated browsers]]</p>
    <![endif]-->
    <noscript>[[Message for users with JavaScript disabled]]</noscript>


    <!-- Insert elements here (from the rest of the guide) to construct your page -->


    <!-- shims.js consists of shims and polyfills for IE8 used in production.
            If IE8 doesn't need to be supported this file can be removed -->
    <!--[if lt IE 9]>
    <script src="scripts/client/shims.js"></script>
    <![endif]-->

    <script src="scripts/client/vendor.js"></script>
    <script src="scripts/client/custom.js"></script>
    <!-- Additional scripts below here -->

  </body>
</html>
    This is the base template you will need. Additions can be made, but you shouldn't remove any of the files unless you are creating a custom build.
  2. Copy the CSS files styles/vendor.css and styles/global.css to your project
  3. Copy the JavaScript files scripts/client/vendor.js and scripts/client/custom.js to your project
  4. Assemble a page by putting together various elements throughout the Pattern Library

Please keep in mind you may need to include additional initialization code listed on the various element pages. This may include instantiating a jQuery plugin, or even including an additional third-party JavaScript file. This code is provided for you on their respective pages.
